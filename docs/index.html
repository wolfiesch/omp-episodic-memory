Why not just use built-in OMP memory?

Oh My Pi's built-in memory is curated and compressed — a distilled view optimized for the agent's working context. That is extremely useful, but it is lossy: the original wording, the dead ends, and the precise moment a decision was made are discarded.

The Curation Gap

When agents compress logs into facts, they throw away the "why" and the verbatim command syntax. When you hit a bug that was solved weeks ago, you need to see the exact logs, code block changes, and CLI outputs — not just a summaries saying "fixed X issue."

Forensic Provenance

Every retrieved result from omp-episodic-memory links directly back to the source session JSONL, user prompt, and assistant code block that solved it. You can track decisions through time, compare options, and review changes on the fly.

Quick Start

From install to your first provenance-backed recall in three commands.

1

Index sessions

Crawl your local Oh My Pi transcripts and build the hybrid SQLite index. Runs fully offline.

$ omp-episodic index
2

Search hybrid

Query across keyword and vector space, fused with RRF. Every hit cites its source session.

$ omp-episodic search "sqlite-vec build"
3

Recall for task

Assemble a confidence-tiered evidence bundle for the task at hand — or abstain when unsure.

$ omp-episodic recall "fix arm64 build"

Core Features

Modern, production-grade capabilities for professional coding environments

Hybrid Retrieval

Fuses Sqlite FTS5 keyword matching and vector matching into a unified ranking system via Reciprocal Rank Fusion (RRF).

Reviewable Memory Inbox

Automatically extracts candidate decisions, gotchas, and runbooks into a staging inbox. You decide what enters the permanent DB.

Evidence Bundles

Retrieved items include confidence metrics and source verification. Correctly abstains instead of fabricating answers.

Temporal Project Graph

Maps relationships over time with automatic supersession rules and diff outputs to check updates since a specific date.

Pinned Context Blocks

Pin durable markdown instructions, rules, or system details to projects to ensure they are always present in the context.

Eval Harness Gates

Comprehensive gold-set tests verifying retrieval precision, abstention recall, and model latency thresholds on CI.
